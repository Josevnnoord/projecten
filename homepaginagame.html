Hallo, ik ben Jose en ik ben gepassioneerd door het bouwen van dingen voor het web. Ik begon mijn reis als ontwikkelaar toen ik een kind was. Ik vond een tutorial over het gebruik van Visual Basic om een bloem op mijn computerscherm weer te geven, en toen dat eenmaal gebeurde, was ik helemaal verslaafd aan de wereld van computers. In de loop der jaren heb ik mijn passie voor coderen vergroot door ITC te bestuderen, maar vooral door games te bouwen met behulp van JavaScript.

Ik ben gepassioneerd door softwareontwikkeling en ik ben bedreven in HTML, JavaScript en PHP. Ik ben nu op zoek naar een uitdaging, omdat ik onlangs zelf heb leren coderen. Zelf heb ik een game gemaakt genaamd, raad het getal klik links bovenin om de game te spelen.
